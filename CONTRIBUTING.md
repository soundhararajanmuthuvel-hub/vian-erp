# Contributing to VIAN ERP

First off, thank you for considering contributing to VIAN ERP!

## Code of Conduct
This project and everyone participating in it is governed by the VIAN Architects developer team guidelines. By participating, you are expected to uphold this code.

## Development Workflow
1. Fork the repository and create your branch from `main`.
2. Ensure you resolve Flutter lints and JS formatting style.
3. Make sure all backend and frontend unit tests compile cleanly.
4. Open a Pull Request detailing the changes, context, and verification steps.
