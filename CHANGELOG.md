# Changelog

All notable changes to the **VIAN ERP** project will be documented in this file.

## [1.0.0] - 2026-07-06

### Added
- **Public Client Enquiry Portal**: Secure public-facing client onboarding wizard with 24 custom sections, site layout sketches canvas, zoom/rotate options, and voice dictation.
- **Client Submission Inbox**: Fully integrated inbox split-panel dashboard allowing admins to view, review notes, approve, and auto-convert enquiries to projects.
- **Auto-Conversion Pipeline**: Automatically creates Client, Project, default construction stages, initial Estimates with BOQ lines, and SiteVisit schedules on submission approval.
- **Security & Route Bypassing**: GoRouter bypass logic for public enquiry routes and token-level verification endpoints.
- **Role Permissions**: Restrict dashboard link generation to MD, Admin, and Tech Head accounts.
